Backups & Data Lifecycle Management

Automated strategies for moving aging files to cold storage and ensuring data redundancy.

Industry Standard Software

For enterprise video teams, dedicated archiving software is the most reliable way to automate data lifecycle management. These tools integrate directly with high-end NAS systems like the OWC Jellyfish.

  • Archiware P5 (Archive): The industry gold standard. Automatically moves files older than a set threshold (e.g., 5 years) to a cheaper NAS or LTO tape, leaving a tiny stub/proxy file in its place.
  • StorageDNA: A heavy hitter in media archiving that automates the movement of old media to LTO tape or cloud storage based on age.
  • Hedge Canister / ChronoSync: Excellent for Mac-based environments to sync and offload older data safely with checksum verification.

Custom Coded Software

If you prefer a custom, subscription-free approach, we will program a Python app that will run system-specific functions for us to sort and back up files automatically every week.

The Workflow:
  1. Trigger: A cron job (Linux/Mac) or Task Scheduler (Windows) runs the script every Sunday at 2:00 AM.
  2. Scan: The script scans the primary "Hot" storage (e.g., Jellyfish) for project folders or files where the "Date Modified" is older than 5 years.
  3. Transfer: It uses rsync or a Python equivalent to securely copy the files to the "Cold" Archive NAS (e.g., 45 Drives XL60).
  4. Verify & Delete: It verifies the checksums of the copied files to ensure data integrity, then deletes the original high-res files from the Hot storage to free up space.
  5. Log: An automated email or Slack/Discord message is sent to the IT/Media Manager with a log of what was moved.

Parsec Remote Desktop

Remote Access for Editors

Parsec is a high-performance remote desktop solution purpose-built for creative professionals. Editors working from home can connect to their on-site editing workstations with near-zero latency, accessing the full power of their local machine and the Jellyfish storage as if they were sitting at their desk.

  • Ultra-low latency: 60fps+ streaming optimized for GPU-accelerated creative apps like Premiere Pro and DaVinci Resolve.
  • Secure connections: End-to-end encryption with team management and access controls.
  • Works on any device: Connect from a laptop, iPad, or secondary PC — no powerful hardware needed remotely.
  • Team plan: ~$20–$35/user/month for business-grade access and admin controls.
Key Advantage: Editors never copy footage to a home machine. All work happens on the on-site workstation connected directly to the Jellyfish — ensuring full storage performance and security even when working remotely.

Strada.tech

High-Speed File Transfer · ~$30/month

Strada.tech is a fast, reliable file transfer platform designed for media professionals who need to send large video files to clients, broadcast partners, or remote collaborators without using consumer services like WeTransfer or Dropbox.

  • Accelerated transfers: Uses UDP-based acceleration (similar to Aspera) to max out available bandwidth regardless of network conditions.
  • Branded portal: Send files from a custom-branded page for a professional client experience.
  • Large file support: No file-size limits — ideal for sending RAW camera footage, master files, or full project packages.
  • Integration-ready: Can be triggered directly from the Axle AI MAM workflow via Connectr™ automation.
Key Advantage: Files are sent directly from the Jellyfish to the recipient — no upload to a third-party cloud required. This keeps your originals on-premise and dramatically speeds up delivery times for large media packages.
